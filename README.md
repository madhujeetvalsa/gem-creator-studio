# Jewelry Studio

A drag-and-drop jewelry design tool. Browse charm pieces, drag them onto the canvas, resize, rotate, and arrange them to create your perfect jewelry layout.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — lint the codebase
- `npm run test` — run unit tests
